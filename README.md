# just_have_fun-Omic_data_analyse

Bioinformatics / omics data analysis project for paper figure reproduction.

## Project overview
This project focuses on reproducing figures and analysis workflows from published omics studies, including proteomics and single-cell transcriptomics.

Current paper:
**Comprehensive single-cell transcriptomic and proteomic analysis reveals NK cell exhaustion and unique tumor cell evolutionary trajectory in non-keratinizing nasopharyngeal carcinoma**

## Objectives
 Reproduce key figures and core analysis workflows from published omics studies
- Organize proteomics and single-cell RNA-seq analysis in a clear and reproducible way
- Improve code readability, annotation quality, and workflow transparency
